# AFilm - Детективная игра о языке камер

Интерактивное расследование, которое учит различать видеокамеры по визуальным характеристикам.

## О проекте

AFilm — это детективная игра, сюжет которой тесно переплетен с характеристиками любительских видеокамер. Мы верим, что каждая камера имеет свой неповторимый визуальный стиль и скрывает в себе много тайн способных дать ответы на вопросы не только в раскрытии нашего преступления, но и в тех обстоятельствах при которых люди и не думают обращаться за решением к технике.

## Концепция

Игра погружает вас в мир производства любительского кино, от тёплых тонов ранней цифровой техники до четкого качества современных смартфонов. Каждый кадр — это улика, которую вы научитесь анализировать с помощью технических характеристик видеокамер.

## Возможности

- **4 эпизода расследования** - Погрузитесь в детективную историю об исчезновении режиссёра
- **Справочник из 6 камер** - Изучите визуальные характеристики каждой камеры
- **Интерактивная игра** - Проверьте свои знания, отвечая на вопросы
- **Система сертификатов** - Получите сертификат после успешного прохождения

1. **Sony DCR-TRV900** (1998) - Тёплая мягкая картинка
2. **Canon XL1** (1998-2001) - Нейтральная документальная цветопередача
3. **Panasonic AG-DVX100** (2002) - Кинематографическая 24p съёмка
4. **Sony DCR-HC1** (2004-2005) - Первая компактная HDV камера
5. **Canon Vixia HF G10** (2010-2012) - Яркая цифровая картинка
6. **GoPro Hero3 Black Edition** (2012) - Сверхширокоугольный объектив

## Запуск проекта

1. Клонируйте репозиторий:
```bash
git clone https://github.com/2014102a-netizen/afilm.git
```

2. Откройте `index.html` в браузере

Или используйте локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server)
npx http-server
```

3. Откройте браузер и перейдите по адресу `http://localhost:8000`

## Как играть

1. Начните с главной страницы `index.html`
2. Нажмите "Начать расследование" или перейдите в раздел "Расследование"
3. Изучите справочник камер для понимания их характеристик
4. Пройдите 4 эпизода, отвечая на вопросы о камерах
5. Получите сертификат детектива после успешного завершения

## Образовательная ценность

AFilm помогает развить:
- **Визуальную грамотность** - Понимание различий в качестве видео
- **Технические знания** - Изучение характеристик видеокамер
- **Критическое мышление** - Анализ визуальных улик
- **Внимание к деталям** - Замечание тонких различий в изображении
